# Styles Directory (`src/styles/`)

This directory manages the master Design System of the application. 

By avoiding disorganized inline-styles and instead piping colors, paddings, and font-families from these singular `theme` configurations, the entire app adheres to strict aesthetics continuously.

## Contents
- **`theme.js`**: The master token registry exposing core `colors` (primary, secondary, accent), typography bindings (Outfit, Roboto), and spacing modifiers.
